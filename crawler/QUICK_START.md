### 1단계: PowerShell 실행
```powershell
cd c:\PRODUCT\SellmeBuyme\crawler

```

### 2단계: 자동 실행
- 모니터 대시보드가 **자동으로 새 창에서 실행**됩니다

- 실시간으로 진행 상황을 확인할 수 있습니다

## 🐛 문제 해결

### 모니터 창이 안 열려요
```powershell
# 수동으로 모니터 실행
cd c:\PRODUCT\SellmeBuyme\crawler
node monitor.js
```

### 이미 실행 중인 백그라운드 크롤러를 모니터링하고 싶어요
```powershell
# 백그라운드 작업의 로그 파일을 지정하여 모니터 실행
node monitor.js --log-file="C:\Users\cgrang\AppData\Local\Temp\claude\c--PRODUCT-SellmeBuyme\tasks\babd0f7.output"
```
- 로그 파일 경로는 백그라운드 작업의 실제 output 파일 경로로 변경하세요
- 모니터가 로그 파일을 실시간으로 파싱하여 진행 상황을 표시합니다

### "진행 상태 파일을 찾을 수 없습니다" 경고
- **정상입니다!** 크롤러가 시작되면 자동으로 파일이 생성됩니다
- `run-all-local.ps1`을 실행하면 자동으로 해결됩니다

### 이모지가 깨져 보여요
1. **Windows Terminal 사용** (cmd.exe는 제한적)
2. **TossFaceFontMac 폰트 설치** (선택사항)
   - `public/fonts/TossFaceFontMac.ttf` 더블클릭 → 설치
   - Windows Terminal 설정에서 폰트 변경

### UI가 비어있어요
- 크롤러가 시작되기 전에는 기본 대기 화면이 표시됩니다
- `R` 키를 눌러 새로고침하세요
- 크롤러가 시작되면 자동으로 업데이트됩니다
