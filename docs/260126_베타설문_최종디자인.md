**작성일**: 2026-01-26
**방식**: 최초 입장 모달 + HeroCard 배너 활용

## 1. 최초 입장 모달 (Welcome Modal)

### 1.1 디자인 목업

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                  학교일자리 베타 서비스                      │
│                 ─────────────────────                      │
│                                                             │
│   현재 테스트 기간을 운영 중입니다.                        │
│   서비스를 이용해보시고 설문에 참여해주시면                │
│   추첨을 통해 스타벅스 커피 쿠폰을 드립니다.               │
│                                                             │
│   ┌──────────────────────────────────────────────┐         │
│   │          설문 참여하기                       │         │
│   └──────────────────────────────────────────────┘         │
│                                                             │
│             [ 나중에 ]                                      │
│                                                             │
│   [ ] 다시 보지 않기                                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 컬러 스킴

- **배경**: 흰색 (#FFFFFF)
- **제목**: 진한 회색 (#1F2937)
- **제목 구분선**: 소프트 레드 (#F87171)
- **본문**: 중간 회색 (#6B7280)
- **설문 버튼**: 파란색 그라디언트 (#5B6EF7 → #3B82F6)
- **나중에 버튼**: 연한 회색 배경 (#F3F4F6), 중간 회색 텍스트 (#6B7280)
- **체크박스**: 연한 회색 (#9CA3AF)

### 1.3 구현 방식

- **표시 타이밍**: 최초 방문 시 즉시 (localStorage 체크)
- **애니메이션**: fade-in + scale (0.95 → 1.0)
- **ESC 키**: 모달 닫기 (= "나중에")
- **배경 클릭**: "나중에" 처리
- **"다시 보지 않기"**: localStorage에 영구 저장

## 2. HeroCard (배너 영역) - 기존 활용

### 2.1 디자인 목업

```
┌────────────────────────────────────────────┐
│  [학교일자리 로고] BETA                     │
├────────────────────────────────────────────┤
│  [학교급 ▼] [과목 ▼]                      │
│  [검색창............................]       │
├────────────────────────────────────────────┤
│  ┌──────────────────────────────────────┐  │
│  │                                      │  │
│  │  학교와 선생님을 찾는               │  │
│  │  가장 쉬운 방법                     │  │
│  │                                      │  │
│  │  설문참여해서 커피쿠폰 받기 →       │  │
│  │                                      │  │
│  └──────────────────────────────────────┘  │
├────────────────────────────────────────────┤
│  공고 목록                      [250개]    │
├────────────────────────────────────────────┤
│  [공고 카드 1]                             │
│  [공고 카드 2]                             │
│  [공고 카드 3]                             │
└────────────────────────────────────────────┘
```

### 2.2 배너 상세

```
┌────────────────────────────────────────────┐
│                                            │
│  학교와 선생님을 찾는                      │  ← 14px, 세미볼드
│  가장 쉬운 방법                            │
│                                            │
│  설문참여해서 커피쿠폰 받기 →              │  ← 12px, 일반
│                                            │
└────────────────────────────────────────────┘
```

### 2.3 컬러 및 스타일

- **배경**: 파란색 (#5B6EF7)
- **텍스트**: 흰색 (#FFFFFF)
- **부제**: 흰색 opacity 80%
- **화살표**: 흰색 opacity 80%
- **호버**: brightness(95%)
- **클릭 시**: 설문 URL 새 탭 열기
- **패딩**:
  - 데스크톱: `px-4 py-4`
  - 모바일: `px-3 py-2.5 md:px-4 md:py-4`

### 2.4 구현 방식

기존 `HeroCard.tsx`의 `DEFAULT_BANNER` 수정:
```typescript
const DEFAULT_BANNER: HeroBanner = {
  id: 'survey',
  title: '학교와 선생님을 찾는 가장 쉬운 방법',
  subtitle: '설문참여해서 커피쿠폰 받기 →',
  bgColor: '#5B6EF7',
  textColor: '#FFFFFF',
  linkUrl: import.meta.env.VITE_SURVEY_URL,
  displayOrder: 0,
  isActive: true,
  createdAt: '',
  updatedAt: ''
}
```

## 9. 주요 특징

1. **이모티콘 없음**: 모든 시각 요소를 텍스트와 화살표(→)로 처리
2. **일관된 컬러**: 기존 사이트의 파란색(#5B6EF7) 테마 유지
3. **소프트 레드**: BETA 배지와 모달 구분선에 #F87171 사용
4. **기존 활용**: HeroCard를 수정하여 재사용
5. **미니멀**: 불필요한 요소 제거, 명확한 계층 구조
