## 1. 사용자 의도 요약

**핵심 문제**: 초등/유치원 기간제교사가 공고를 찾을 때 **방과후 강사, 돌봄, 실무직 공고가 섞여서** 불편함

**해결 목표**: 학교급별로 **직종 필터**를 추가하여 원하는 공고만 볼 수 있게 함

## 3. 핵심 인사이트

### 3.1 초등학교 실제 상황

**기간제교사 찾는 사람 입장**:
- 원하는 것: `기간제-담임(44)` + `기간제-전담(20)` + `기간제-일반(231)` = **약 295건**
- 안 보고 싶은 것: 방과후/늘봄(196) + 외부강사(157) + 돌봄(76) = **약 429건**

→ **현재 섞여 있어서 절반 이상이 관심 없는 공고**

**방과후 강사 찾는 사람 입장**:
- 원하는 것: `방과후/늘봄(196)` + `외부강사(157)` + `돌봄(76)` = **약 429건**
- 분류 필요: 체육, 음악, 미술, 코딩 등 과목별

### 3.2 유치원 실제 상황

**기간제교원 찾는 사람 입장**:
- 원하는 것: `기간제-일반(91)` + `기간제-특수/통합(26)` = **약 117건**
- 안 보고 싶은 것: 방과후(309) + 특성화강사(104) = **약 413건**

→ **유치원은 방과후/특성화가 54%로 압도적**

## 5. 실제 분류 로직 (SQL WHERE 조건)

### 5.1 초등 - 기간제교원

```sql
WHERE school_level = '초등'
  AND (
    title LIKE '%기간제%'
    OR title LIKE '%계약제%'
  )
  AND title NOT LIKE '%방과후%'
  AND title NOT LIKE '%늘봄%'
  AND title NOT LIKE '%외부강사%'
```

### 5.2 초등 - 방과후/늘봄

```sql
WHERE school_level = '초등'
  AND (
    title LIKE '%방과후%'
    OR title LIKE '%늘봄%'
    OR title LIKE '%개인위탁%'
    OR title LIKE '%외부강사%'
  )
```

### 5.3 초등 - 돌봄

```sql
WHERE school_level = '초등'
  AND title LIKE '%돌봄%'
  AND title NOT LIKE '%방과후%'
```

### 5.4 초등 - 실무직

```sql
WHERE school_level = '초등'
  AND (
    title LIKE '%실무%'
    OR title LIKE '%조리%'
    OR title LIKE '%시설관리%'
    OR title LIKE '%미화%'
    OR title LIKE '%영양사%'
  )
```

## 7. 구현 우선순위

### Phase 1 (핵심)
1. **초등 2분류**: 기간제교원 vs 방과후/늘봄/강사
2. **유치원 2분류**: 기간제교원 vs 방과후/특성화

→ 이것만 해도 사용자 불편 70% 해결

### Phase 2 (세분화)
1. 초등 기간제 세분류 (담임/전담/보건/특수/상담/사서/영양)
2. 초등 방과후 과목분류 (체육/음악/미술/코딩)
3. 실무직 분류 (조리/행정/시설)

### Phase 3 (선택)
1. 기타/교육청 분류
2. 자원봉사 필터
