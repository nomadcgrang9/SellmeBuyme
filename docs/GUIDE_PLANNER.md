## 🔧 주요 기능 목록

### 1. AI 맞춤 추천

```
사용자가 프로필 등록
       ↓
AI가 다단계 점수 계산
       ↓
상위 6개 공고 추천 + AI 코멘트 생성
```

#### 점수 계산 기준 (상세)

**1. 지역 점수**
- 관심 지역 직접 매칭: +8점
- **인접 지역 자동 확장**: 수원 → 용인, 화성, 의왕, 오산 포함
- 인접 지역 맵: 서울↔고양/광명/구리/과천/성남/부천 등 25개 연결

**2. 역할 기반 분류**
| 역할 | 우대 공고 | 패널티 공고 |
|------|---------|-----------|
| 교사 | 수업 관련 (+25) | 급식/행정 (-40) |
| 강사 | 수업 관련 (+12) | 급식/행정 (-15) |
| 학교행정 | 급식/행정/봉사 (+28) | 수업 관련 (-12) |

**3. 교과 호환성 검사**
- 초등 담임 → 초등 모든 공고 매칭
- 중등 과목 → 해당 과목 + 초등 전담 가능 (상향 호환)
- 중등 물리 → "과학" 공고도 매칭

**4. 마감 임박 가중치**
| 마감까지 | 가산점 |
|---------|-------|
| 1일 이내 | +50점 |
| 2일 이내 | +35점 |
| 3일 이내 | +20점 |
| 7일 이내 | +8점 |
| 마감 지남 | -100점 (제외) |

**5. 최신성**
- 3일 이내 공고: +3점
- 2주 초과 공고: -30점

**6. AI 2차 필터링 (Gemini)**
- 상위 60개 후보 중 8~12개 최종 선별
- 지역 다양성 고려 (한 지역 최대 3개)

### 2. 검색/필터
- **자연어 검색**: "수원 초등 방과후 코딩"
- **필터 검색**: 지역, 학교급, 과목, 정렬 기준
- **동의어 확장**: "중등" 검색 시 "중학교", "고등학교"도 포함

### 3. 카드 시스템

| 카드 타입 | 색상 | 설명 |
|----------|------|------|
| **일자리** | 파란색 (`#a8c5e0`) | 채용 공고 |
| **인력** | 초록색 (`#c5e3d8`) | 강사/교사 프로필 |
| **체험** | 노란색 (`#ffd98e`) | 체험 프로그램 |

### 4. 북마크
- 하트 아이콘으로 공고/인력 저장
- 모달 또는 페이지에서 목록 확인

### 5. 채팅
- 공고 작성자 ↔ 지원자 실시간 대화
- 카카오톡처럼 즉시 전달

### 6. 등록 기능

| 등록 유형 | 등록자 | 주요 입력 항목 |
|----------|--------|--------------|
| 일자리 등록 | 학교 | 조직명, 직책, 지역, 급여, 마감일 |
| 인력 등록 | 강사 | 이름, 전문분야, 경력, 자격증 |
| 체험 등록 | 교육기관 | 프로그램명, 대상, 운영방식 |

### 7. 프로필 설정 (3단계)

Google 로그인 후 처음 방문 시 표시되는 프로필 설정 과정입니다.

#### Step 1: 역할 선택
> "어떤 역할로 활동하세요?"

| 역할 | 설명 |
|------|-----|
| **교사** | 공고를 직접 등록하거나 기간제 교사로 활동 |
| **강사** | 교사와 함께 협력수업 또는 방과후 단독수업, 교직원·학부모 대상 강의 |
| **업체** | 개발한 프로그램이나 커리큘럼을 학교에 적용하고 비용을 받음 |
| **학교행정** | 자원봉사자, 조리사, 행정인력 대체 등 교육활동 지원 |

#### Step 2: 분야 선택
> "어떤 학교급에서 활동하세요?" / "강사 분야를 선택해 주세요"

**교사인 경우:**
- 학교급 선택: 유치원, 초등, 중등, 특수
- 고용 형태 선택: 기간제 교사 / 정규 교원
- (중등일 때) 담당 가능 교과 선택: 국어, 수학, 과학, 영어, 체육, 음악, 미술 등 17개 과목 + 직접 입력
- 초등 전담 가능 과목도 선택 가능 (중등 → 초등 호환)

**강사인 경우:**
- 분야 선택: 요리, 코딩, 음악, 체육, AI교육, 심리상담, 교권보호, 유아놀이, 미술교육, 독서코칭
- 추가 분야 직접 입력 가능

#### Step 3: 지역 + 자기소개
> "활동하고 싶은 지역을 선택해주세요"

- **서울**: 전체 선택 또는 24개 구 개별 선택
- **경기도**: 전체 선택 또는 31개 시/군 개별 선택
- 자기소개 입력 (최대 500자)
  - 예) "초등담임 기간제 가능하고 생활지도 업무 다수 경력 있습니다. 또한 교직원 및 학부모 대상 교권보호 강의도 가능합니다."

## 📈 관리자 기능

### 관리자 대시보드 (`/admin` 또는 `/admin-portal`)

| 탭 | 기능 | 설명 |
|-----|------|-----|
| **대시보드** | 요약 지표 | 오늘 신규 공고, 사용자 수, 크롤링 상태 등 |
| **크롤링 게시판 관리** | 크롤러 소스 관리 | 개발자 제출 승인, 게시판 추가/수정, 로그 확인 |
| **홍보카드 관리** | 배너 편집 | 추천 섹션 프로모카드 + 띠지 배너 편집 |
| **사용자 관리** | 회원 목록 | 가입 사용자 목록 및 프로필 조회 |
| **콘텐츠 관리** | 공고/인력 검수 | 등록된 공고와 인력 검토 |
| **설정** | 시스템 설정 | 권한 및 시스템 설정 |

#### 크롤링 게시판 관리 상세
- **보드 제출 목록**: 개발자가 제출한 새 크롤러 보드 승인/거절
- **게시판 목록**: 활성 크롤러 관리 (일시정지, 배치 크기 변경)
- **크롤 로그 뷰어**: 각 보드별 크롤링 성공/실패 이력

### 개발자노트 (`/note`) - PWA

| 섹션 | 기능 | 설명 |
|------|-----|-----|
| **아이디어** | 기능 제안 관리 | 아이디어 등록, 상태 추적 (제안→검토→개발→완료) |
| **보드 제출** | 크롤러 보드 제출 | 새 교육청 게시판 크롤러 제출 (관리자 승인 필요) |
| **배포 현황** | 배포 이력 | Cloudflare Pages 배포 기록 추적 |
| **에러 로그** | 에러 모니터링 | 모바일/PC 에러 발생 현황 확인 |

## 🗂 주요 데이터 항목

### 일자리 공고 (job_postings)
| 필드 | 설명 | 예시 |
|------|-----|-----|
| 조직명 (organization) | 학교/기관 이름 | "서울초등학교" |
| 제목 (title) | 공고 제목 | "2024학년도 2학기 영어전담 기간제교사 채용" |
| 지역 (location) | 근무지 | "경기도 수원시 영통구" |
| 급여 (compensation) | 보상 정보 | "호봉제 적용" |
| 마감일 (deadline) | 모집 마감일 | "2024-12-15" |
| 학교급 (school_level) | 학교 수준 | 유치원/초등/중등/특수 |
| 과목 (subject) | 담당 과목 | "영어", "과학" |
| 긴급 여부 (is_urgent) | 급히 구함 표시 | ✓/✗ |
| 태그 (tags) | 키워드 태그 | ["기간제교사", "영어", "초등"] |
| 근무 기간 (work_period) | 근무 시작~종료 | "2024.09.01 ~ 2025.02.28" |
| 접수 기간 (application_period) | 지원 가능 기간 | "2024.08.01 ~ 2024.08.15" |
| 근무 시간 (work_time) | 하루 근무시간 | "09:00 ~ 16:00" |
| 연락처 (contact) | 담당자 연락처 | "031-123-4567" |
| 원문 URL (source_url) | 원본 공고 링크 | 교육청 공고 페이지 |
| 첨부파일 (attachment_url) | 공고문 파일 | PDF 다운로드 링크 |
| 출처 (source) | 수집 경로 | "crawled" / "user_posted" |

|------|-----|

| 전문분야 | 주요 전공/과목 |
| 지역 | 활동 가능 지역 |
| 경력 | 경력 연수 |
| 평점 | 리뷰 평점 |
| 자격증 | 보유 자격 |
| 검증 여부 | 검증된 강사 표시 |

### 체험 프로그램 (experiences)
| 필드 | 설명 |
|------|-----|
| 프로그램명 | 체험 이름 |
| 운영방식 | 현장/온라인/하이브리드 |
| 대상 학교급 | 타겟 학년 |

### 사용자 프로필 (user_profiles)
| 필드 | 설명 | 예시 |
|------|-----|-----|
| 표시 이름 (display_name) | 닉네임/이름 | "김선생" |
| 역할 (roles) | 활동 역할 (복수 선택) | ["교사", "강사"] |
| 관심 지역 (interest_regions) | 선호 근무 지역 | ["수원", "용인", "화성"] |
