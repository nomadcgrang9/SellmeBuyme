## 🔧 주요 기능 목록

### 1. AI 맞춤 추천
```
사용자가 프로필 등록
       ↓
AI가 5가지 기준으로 점수 계산
  • 위치 (같은 구 > 같은 시 > 인접 도시)
  • 역할 (교사/강사 매칭)
  • 과목 (가능한 과목 일치)
  • 경력 (경력 연차)
  • 긴급도 (마감 임박 우선)
       ↓
상위 6개 공고 추천
```

### 2. 검색/필터
- **자연어 검색**: "수원 초등 방과후 코딩"
- **필터 검색**: 지역, 학교급, 과목, 정렬 기준
- **동의어 확장**: "중등" 검색 시 "중학교", "고등학교"도 포함

### 3. 카드 시스템

| 카드 타입 | 색상 | 설명 |
|----------|------|------|
| **일자리** | 파란색 (`#a8c5e0`) | 채용 공고 |
| **인력** | 초록색 (`#c5e3d8`) | 강사/교사 프로필 |
| **체험** | 노란색 (`#ffd98e`) | 체험 프로그램 |

### 4. 북마크
- 하트 아이콘으로 공고/인력 저장
- 모달 또는 페이지에서 목록 확인

### 5. 채팅
- 공고 작성자 ↔ 지원자 실시간 대화
- 카카오톡처럼 즉시 전달

### 6. 등록 기능

| 등록 유형 | 등록자 | 주요 입력 항목 |
|----------|--------|--------------|
| 일자리 등록 | 학교 | 조직명, 직책, 지역, 급여, 마감일 |
| 인력 등록 | 강사 | 이름, 전문분야, 경력, 자격증 |
| 체험 등록 | 교육기관 | 프로그램명, 대상, 운영방식 |

### 7. 프로필 설정 (5단계)
1. **기본 정보**: 이름, 역할 선택
2. **학력**: 교육 수준
3. **전공/과목**: 가능한 과목
4. **거주지**: 선호 지역
5. **스킬/자격증**: 보유 자격

## 📊 데이터 흐름

### 데이터는 어디서 오나요?

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   [자동 수집]                    [직접 등록]                  │
│                                                             │
│   25개 교육청 게시판              사용자가 앱에서 등록         │
│         ↓                              ↓                    │
│   크롤러 (매일 아침 7시)                ↓                    │
│         ↓                              ↓                    │
│   AI가 내용 분석                       ↓                    │
│         ↓                              ↓                    │
│         └──────────────┬───────────────┘                    │
│                        ↓                                    │
│                   데이터베이스                                │
│                        ↓                                    │
│                   검색/추천                                   │
│                        ↓                                    │
│                   사용자 화면                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 크롤러 동작 과정

```
1. 교육청 게시판 접속
       ↓
2. 공고 목록 화면 캡처
       ↓
3. AI가 이미지에서 정보 추출
   "학교명: OO초, 과목: 영어, 급여: 시급 3만원..."
       ↓
4. 이미 있는 공고인지 확인 (중복 제거)
       ↓
5. 데이터베이스에 저장
       ↓
6. 사용자들이 검색 가능
```

## 🎨 UI/UX 구성 요소

### 헤더 (Header)
- 로고
- 검색창
- 필터 버튼
- 사용자 메뉴 (로그인/프로필)

### 카드 그리드 (CardGrid)
- 무한 스크롤
- 3가지 타입 (일자리/인력/체험)
- 호버 시 액션 버튼 표시

### 바텀 네비게이션 (모바일)
- 홈
- 탭 전환 (일자리/인력/체험)
- 등록 (+)
- 채팅
- 프로필

### 모달 종류
| 모달 | 용도 |
|------|-----|
| 소셜 로그인 | Google 로그인 |
| 프로필 설정 | 5단계 프로필 입력 |
| 프로필 보기 | 프로필 조회/편집 |
| 상세 보기 | 공고/인력/체험 상세 정보 |
| 북마크 | 저장된 항목 목록 |
| 채팅 | PC에서 채팅 (모달) |
