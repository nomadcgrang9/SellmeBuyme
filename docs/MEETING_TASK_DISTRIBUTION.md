# 🗺️ 10분 지역별 공고 매칭 점검 (R&R)

## 🎯 목표
각 교육청 공식 홈페이지의 **최신 공고(1번 게시글 기준)**가 실제 우리 서비스(지도)에 정상적으로 노출되는지 확인합니다.
- **소요 시간**: 10분
- **확인 대상**: 각자 할당된 지역의 교육청 구인 게시판

## ✅ 체크 포인트
1. **게시판 접속**: 위 링크를 클릭하여 교육청 게시판에 접속합니다.
2. **최신 글 확인**: 게시판의 **가장 최신 공고(1번)**를 클릭합니다. (공지사항 제외. 최근 1페이지 내의 게시글이면 됩니다)
3. **서비스 대조**: 우리 서비스(지도)에서 해당 지역을 찾아, 같은 공고가 지도상에 표시되는지 확인합니다.
4. **피드백**: "배정된 지역 OOO, 1번 공고 지도에 (있음/없음)" 형태로 채팅방에 남겨주세요.
