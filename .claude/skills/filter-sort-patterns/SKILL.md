# Filter Sort Patterns

## Description
Designs filtering and sorting interfaces for data exploration. Use when creating filter panels, sort controls, or when the user asks about data filtering UX.

## Instructions

### Filter Types

| Type | UI Pattern | Best For |
|------|------------|----------|
| **Dropdown** | Single/multi-select | Few options (5-10) |
| **Checkbox Group** | Multiple selection | Related options |
| **Range Slider** | Numeric ranges | Price, distance, date |
| **Search Filter** | Text matching | Large option sets |
| **Toggle** | Boolean on/off | Binary options |
| **Chips** | Tag selection | Visual, touch-friendly |

### URL State Sync

Preserve filter state in URL:
```
/jobs?school=elementary&region=seoul&sort=deadline
```

Benefits:
- Shareable filtered views
- Back button works correctly
- Bookmarkable searches
- SEO friendly

## Best Practices

1. Show results count immediately
2. Remember user's filter preferences
3. Provide sensible defaults
4. Design mobile-first filter UX
5. Fast filter response (< 300ms perceived)
6. Debounce text input filters
