# 260129 교원연수 강사등록 시스템 구현 플랜

## 1. 개요 및 배경
기존의 '인력 등록' 시스템은 주로 **학생 대상**의 수업(기간제, 방과후, 기초학력 등)을 위한 강사 풀을 형성하는 데 초점이 맞춰져 있었습니다.
반면, 이번에 구현할 **'교원연수 강사등록'**은 **성인(교원, 학부모, 교육행정직 등)**을 대상으로 전문적인 연수나 강의를 제공할 수 있는 전문가를 모집하는 시스템입니다.

### 주요 차이점
| 구분 | 기존 등록 시스템 | 신규 교원연수 강사등록 |
| :--- | :--- | :--- |
| **주 타겟** | 유/초/중/고 학생 | 교원, 학부모, 교육공무직 |
| **활동 성격** | 수업, 돌봄, 생활지도 | 직무연수, 소양교육, 힐링프로그램, 전문가 특강 |
| **자격 요건** | 교원자격증, 강사자격 | 특정 분야 전문가, 석박사, 전문 강사 자격 |
| **필요 정보** | 학교급, 과목, 경력(년수) | **강의 주제(카테고리), 강의 이력, 출강 가능 분야** |

## 3. UI/UX 구현 방안

### 3-1. 등록 모달 창 (Registration Modal)
기존 `TalentRegistrationForm`을 확장하거나 **별도의 모드**로 분리하여 구현합니다.
가장 상단에서 **"등록 유형 선택"** 탭을 두어 사용자 경험을 분리합니다.

*   **탭 구성**: `[ 학생 대상 강사 등록 ]` vs `[ 교원/학부모 연수 전문가 등록 ]`
*   **교원연수 등록 시 입력 폼**:
    1.  **전문 분야 선택 (필수)**: 위 20개 카테고리를 그리드 형태의 체크박스로 제공 (복수 선택 가능).
    2.  **강의 대상 (복수 선택)**: `교원`, `학부모`, `행정직원`, `기타`
    3.  **대표 강의명**: 자신의 대표 강의 주제 1~2줄 요약 (예: "생성형 AI를 활용한 업무 자동화")
    4.  **주요 이력/자격**: 관련 학위, 저서, 자격증, 주요 출강 이력 텍스트 에어리어.
    5.  **활동 지역**: 서울/경기 (기존과 동일).

### 3-2. 필터바 (Filter Bar Update)
메인 지도 화면 하단 `CascadingFilterBar` 및 `jobClassifier.ts` 수정이 필요합니다.

1.  **1차 카테고리 (Primary)**: `교원/학부모 연수` (신규 추가)
    *   기존: 유치원, 초등담임 ... 행정/교육지원, [ **교원/학부모 연수** ]
    *   색상 테마: 신뢰감을 주는 색상 (예: `Indigo` 또는 `Deep Purple`)
2.  **2차 카테고리 (Secondary)**:
    *   1차에서 '교원/학부모 연수' 선택 시, 위 20개 카테고리가 칩 형태로 나열됨.
    *   양이 많으므로 가로 스크롤 또는 2줄 배치 고려.

## 4. 데이터베이스 및 로직 설계

### 4-1. DB 스키마 (Supabase)
`profiles` 테이블 또는 별도의 `instructor_profiles` 테이블에 다음 컬럼을 추가/활용합니다.

| 필드명 | 타입 | 설명 |
| :--- | :--- | :--- |
| `is_adult_training` | `boolean` | 성인 대상 연수 강사 여부 |
| `adult_training_categories` | `text[]` (Array) | 선택한 카테고리 목록 (예: `['AI·디지털', '행정·리더십']`) |
| `target_audience` | `text[]` (Array) | 주요 강의 대상 (`['teacher', 'parent']`) |
| `lecture_history` | `text` | 주요 강의 이력 및 자격 상세 (Self-introduction과 별도 관리 권장) |

### 4-2. 로직 (`jobClassifier.ts`)
*   `PRIMARY_CATEGORIES`: `adult_training` 키 추가.
*   `SECONDARY_OPTIONS`: `adult_training` 키에 20개 리스트 매핑.
*   `classifyJob` 함수: 구인 공고(나중에 연수 강사 구인 시)에서도 제목에 '연수', '특강', '부모교육' 등이 포함되면 자동으로 이 카테고리로 분류되도록 로직 추가.
